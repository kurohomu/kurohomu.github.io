Scalar and Vector Field

A scalar or vector quantity is said to be a field if it is a function of position.

Scalar Field

In mathematical terms, a Scalar field is a function of the variables $x, y, z$ (in [[Fields/Coordinate System#Cartesian system | Cartesian coordinates]]) that defines a scalar (such as pressure, temperature, electrostatic potential, etc.) at each point in [[Fields/Coordinate System | spcae]] (or in a region of space).

Level Surface

The level surfaces of a given scalar field $G(x, y, z)$ are those surfaces on which the scalar field has a constant value, that is,
$$
G(x, y, z)=k
$$
where $k$ is a constant value on the surface.

There are some special scalar field:

  • isotherms if $G$ represents temperature,
  • isobars if $G$ represents pressure,
  • equipotentials if $G$ represents a scalar potential.

Vector Field

In mathematical terms a vector field is a vector whose components are functions of the variables $x, y$ and $z$ in [[Fields/Coordinate System#Cartesian system | Cartesian coordinates]].

A vector field is often denoted using the following notations $\mathbf{A}(x, y, z), \mathbf{v}(x, y, z)$, or $\mathbf{F}(x, y, z)$
$$
\mathbf{A}(x, y, z)=A_{x}(x, y, z) \hat{\mathbf{i}}+A_{y}(x, y, z) \hat{\mathbf{j}}+A_{z}(x, y, z) \hat{\mathbf{k}}
$$
The vectors $\hat{i}, \hat{j}, \hat{j}$ are unit vectors in the $x, y$ and $z$ directions, respectively.

Scalar and vector fields in other coordinate systems

Since scalar and vector fields are functions of position, then their functionality will change depending on the coordinate system used. In cylindrical polar coordinates, the scalar quantity would be a function of $r, \theta$ and $z$, i.e, $G=G(r, \theta, z)$ while in spherical polar coordinates $G=(\rho, \phi, \theta)$
For the corresponding vector $\mathbf{A}$, in cylindrical polar coordinates
$$
\mathbf{A}(r, \theta, z)=A_{r}(r, \theta, z) \hat{\mathbf{r}}+A_{\theta}(r, \theta, z) \hat{\boldsymbol{\theta}}+A_{z}(r, \theta, z) \hat{\mathbf{z}}
$$
while in spherical polar coordinates
$$
\mathbf{A}(\rho, \phi, \theta)=A_{\rho}(\rho, \phi, \theta) \hat{\boldsymbol{\rho}}+A_{\phi}(\rho, \phi, \theta) \hat{\boldsymbol{\phi}}+A_{\theta}(\rho, \phi, \theta) \hat{\boldsymbol{\theta}}
$$

Additional definitions in field analysis

Consider a vector field $\mathbf{A}=\mathbf{A}(x, y, z, t)$.

  • This field is steady, if $\mathbf{A}=\mathbf{A}(x, y, z)$, i.e., the vector field $\mathbf{A}$ is not dependent on $t$, in turn meaning $\frac{\partial \mathbf{A}}{\partial t}=0$.
  • The field $\mathbf{A}$ is unsteady, if $\frac{\partial \mathbf{A}}{\partial t} \neq 0$.
  • The field $\mathbf{A}$ is uniform, if it is independent of the space coordinates, that is
    $$
    \frac{\partial \mathbf{A}}{\partial x}=\frac{\partial \mathbf{A}}{\partial y}=\frac{\partial \mathbf{A}}{\partial z}=\mathbf{0} \quad(\mathbf{A}=\mathbf{A}(t))
    $$
  • A field is said to be one-dimensional, if it is a function of only one space coordinate and time, for example, $\mathbf{A}=\mathbf{A}(x, t)$.
  • A two-dimensional field is a function of two space coordinates and time, such as $\mathbf{A}(x, y, t)$.
  • A three-dimensional field depends on three space coordinates and time.

Notion: the justify of steady is the partial derivative, but not the material derivative.
