Green’s Theorem

The Green’s theorem provide a relation of [[Fields/Integral/Curve integral|integral]] between closed curve and region over it in 2D.

#Theorem
Let $R$ be a [[Complex Analysis/Connected Region#Simple Connected Region|simply connected plane region]] whose boundary is a simple piecewise smooth closed curve $C$ oriented counterclockwise (anticlockwise). If $f(x, y)$ and $g(x, y)$ have continuous first partial derivatives on some open set containing the region $R$, then$$\int_{C} f(x, y) d x+g(x, y) d y=\iint_{R}\left(\frac{\partial g}{\partial x}-\frac{\partial f}{\partial y}\right) d A$$

Proof

$\begin{aligned} \int_{C} f(x, y) d x &=\int_{C_{1}} f(x, y) d x-\int_{-C_{2}} f(x, y) d x \ &=\int_{a}^{b} f\left(t, p_{1}(t)\right)\left(\frac{d x}{d t}\right) d t-\int_{a}^{b} f\left(t, p_{2}(t)\right)\left(\frac{d x}{d t}\right) d t \ &=\int_{a}^{b} f\left(t, p_{1}(t)\right)\left(\frac{d t}{d t}\right) d t-\int_{a}^{b} f\left(t, p_{2}(t)\right)\left(\frac{d t}{d t}\right) d t \ &=\int_{a}^{b} f\left(t, p_{1}(t)\right) d t-\int_{a}^{b} f\left(t, p_{2}(t)\right) d t \ &=-\int_{a}^{b}\left[f\left(t, p_{2}(t)-f\left(t, p_{1}(t)\right] d t\right.\right.\ &=-\int_{a}^{b}[f(t, y)]{y=p{1}(t)}^{y=p_{2}(t)} d t \ &=-\int_{a}^{b}\left(\int_{p_{1}(t)}^{p_{2}(t)} \frac{\partial f}{\partial y} d y\right) d t \ &=-\int_{a}^{b} \int_{p_{1}(x)}^{p_{2}(x)} \frac{\partial f}{\partial y} d y d x \ &=-\int_{R} \frac{\partial f}{\partial y} d A \end{aligned}$
