Coordinate System

A coordinate system is a means by which every position in space can defined or determined uniquely.
To study a system, we need its triple, restriction, graph, constant surface, interchange. The restrictions on the values taken in the cylindrical and spherical coordinates ensure that each point, other than the origin, has a unique set of coordinates.
